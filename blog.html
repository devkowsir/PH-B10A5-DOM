What is DOM?

The Document Object Model is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document.

How do you select an element from the DOM?

The document api has multiple methods for selecting element. Most commonly used are `getElementById`, `querySelector`, `querySelectorAll`. Less commonly used are `getElementsByClassName`, `getElementsByName`, `getElementsByTagName`.

What is event delegation in the context of the DOM and why it should be used?

We often need to listen for the same event in different elements. As we know, events has to go throught capture and bubble phases. In the above case we simply can just use one listener on the parent of those elements and handle the filtering logic there. This is called event delegation. This reduces number of event listeners.

How do you manipulate an element's attributes and styles using the DOM?

The document api gives us mutiple ways of doing this. We have `setAttribute` method for setting attributes(as well as style). For style we can use `style` property. We can also manipulate style by changing element classNames by the `classList` property which gives use `add`, `remove`, `toggle` methods.
